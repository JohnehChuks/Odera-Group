Phased Array Ultrasonic Testing

Phased Array Ultrasonic Testing (PAUT), also known as phased array UT, is an advanced non-destructive inspection technique that uses a set of ultrasonic testing (UT) probes made up of numerous small elements. Each of these is pulsed individually with computer-calculated timing to create the phased aspect of the process, while the array refers to the multiple elements that make up a PAUT system.

Phased Array Ultrasonic Testing: Level 2

This programme provides detailed instructions on the inspection of ferritic welds/castings/wrought products by the use of phased array ultrasonic testing. Suitable only for holders of current, valid UT certification approved and accepted by PCN.

Phased Array Ultrasonic Testing: Level 3

This is a guidance course aimed at the PCN requirements for Ultrasonic Phased Array Practitioner Level 3 of general engineering materials, components and fabrications in the pre and in-service inspection multi sector. Alternatively Level 3 candidates may apply for single-sector certification covering one (only) of the following groups: Welds, Castings or Wrought Products.
